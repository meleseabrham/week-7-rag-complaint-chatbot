# RAG Complaint Chatbot

A Retrieval-Augmented Generation (RAG) based chatbot for handling complaints.

## Project Structure

- `data/`: Raw and processed data.
- `vector_store/`: Persisted vector database (FAISS/ChromaDB).
- `src/`: Source code.
- `notebooks/`: Research and development.
- `tests/`: Unit and integration tests.
- `app.py`: Gradio/Streamlit application.
